# graphics

## Dependencies
### Windows
- CMake
- Visual Studio
- Vulkan SDK
### Linux
- CMake
- Vulkan SDK
- Slang
### macOS
- CMake

## Usage:
1. Create directory for building ` Build `
2. Run CMake on the root directory and the build directory ` cmake -S <source> -B <build> `
3. Choose the generator ` cmake -G <generator> `
4. Compile the build directory
