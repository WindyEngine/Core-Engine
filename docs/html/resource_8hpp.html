Detailed Description

Declares the core classes and interfaces for resource management within the engine.

This file contains declarations for the Resource, ResourceLoader, and ResourceEvent classes used for managing, loading, unloading, and hot-reloading various engine resources. It provides facilities for dependency tracking, change propagation, and observer pattern implementation to handle resource-related events.
