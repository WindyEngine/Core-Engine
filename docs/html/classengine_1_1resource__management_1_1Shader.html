Detailed Description

Represents a shader resource that can be compiled into multiple target shading languages.

The Shader class is a concrete subclass of Resource and is responsible for managing shader source code. It supports loading from disk, unloading, and saving operations, and integrates with the engine's shader compiler to produce output in various shading languages, including:

  • HLSL (High-Level Shading Language, used in DirectX)
  • GLSL (OpenGL Shading Language)
  • MSL (Metal Shading Language, used on Apple platforms)
  • SPIR-V (Standard Portable Intermediate Representation for Vulkan)
