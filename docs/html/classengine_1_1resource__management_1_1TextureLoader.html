Detailed Description

Responsible for instantiating and initializing Texture resources.

The TextureLoader class is a concrete implementation of the abstract ResourceLoader interface. It is responsible for creating and loading Texture objects from file paths, and optionally handling lazy-loading strategies.

This loader is registered with the resource manager and is invoked automatically when a .png, .jpg, .dds, or other supported texture file is requested.
