Detailed Description

template<typename T>
class engine::resource_management::ResourceHandle< T >

A lightweight handle to a managed resource.

This class wraps a std::shared_ptr to provide safe, convenient access to resources without exposing ownership details.

Template Parameters
TThe type of the resource being handled.
